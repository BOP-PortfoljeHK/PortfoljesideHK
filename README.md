# Glass artist Portfolio Website


## Project Overview

This project is a **digital portfolio website** developed for a glass artist, with the goal of showcasing their work, skills, and artistic identity online.
The project is also part of the course **BOP3000 (Bach IT og inf.syst 26V Bø)**, as the final project in the bachelors degree.

The solution focuses on creating a **visually appealing and easy-to-manage platform**, allowing the client to update content independently without technical knowledge.

---

## Background

Artists working with handcrafted products rely heavily on visual presentation. This project addresses the need for a **professional digital portfolio** where the client can:

- Showcase their artwork  
- Present their creative identity  
- Maintain and update content independently  

The project does not include e-commerce functionality, as the primary focus is presentation and content management.

---

## Features

- Responsive and modern portfolio design  
- Dynamic content management via CMS  
- Image-focused layout for showcasing artwork  
- Easy content updates without developer involvement  
- Clean and minimal UI for highlighting visual work  

---

## Tech Stack

###  Frontend
- **Astro** – Static site generation and performance-focused architecture  
- **Tailwind CSS** – Utility-first styling for rapid UI development  

### CMS / Backend
- **Sanity** – Headless CMS for managing portfolio content  

### Deployment
- **Netlify** – Hosting and continuous deployment  

### Development Tools
- **Git & GitHub** – Version control  
- **Visual Studio Code** – Development environment  

---

## Key Concepts

- Static site generation (SSG)  
- Headless CMS architecture  
- Component-based UI design  
- Content-driven development  
- Separation of content and presentation  

---

## Development Process

The project focused on delivering a **user-friendly and maintainable solution**:

- Designed with the end-user (artist) in mind  
- Emphasis on simplicity and usability  
- Iterative development with continuous feedback  
- Structured for easy future expansion
